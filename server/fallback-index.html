Daily Team Whisper API

Welcome to the Daily Team Whisper API. This is the backend server for the AutoBrief application.

Available Endpoints

GET /api/integrations

Get user's integrations

GET /api/activities

Get user's work activities

GET /api/summaries

Get user's daily summaries

POST /api/ai/daily-summary

Generate AI daily summary

Development Mode

The server is running in development mode. The React frontend is available at http://localhost:3000.

Health Check

You can check the server status at /health.
